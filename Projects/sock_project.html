Developing an Interactive List-Based Sock Retailer

About this Project

The goal of this application was to develop an interactive interface, use interface components, and tie the components to an internal data state. I wanted to create a Sock selling website where you could add items of interest to your cart, remove them easily, filter them by categories, and sort them by price. This was intended to be a project to help expose myself to the react library and grasp a better understanding of it's powerful components structure.
