Redesigning an Interactive Interface

About this Project

The aim of this application was to practice the workflow of redesigning a simple website due to learnability, memorability, or efficiency issues. In other words, is the website easy for new users to navigate through, can tasks be done quickly, and is it easy to recall what to do. The website, Manganato , was picked due to the connection I have with it as a regular user and the passion I have towards improving the interface.

Some valuable skills picked up throughout this process was analyzing and identifying flaws in an existing interface, creating low-fidelity and high-fidelity prototypes for various screen sizes, and building a responsive website based on those prototypes.
Timeline 2 weeks
